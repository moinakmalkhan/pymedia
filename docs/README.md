# Documentation Structure

This folder contains the source markdown files for project documentation.

## Entry Point

- `index.md`: Main landing page with section links.

## API References

- `video.md`: Full video API reference.
- `audio.md`: Full audio API reference.
- `frames.md`: Frame extraction and preview reference.
- `metadata.md`: Metadata read/write utilities.
- `info.md`: Media probing reference.
- `analysis.md`: Timeline and analysis utilities.
- `subtitles.md`: Subtitle conversion and track operations.
- `streaming.md`: Streaming-oriented and packaging-oriented APIs.

## Guides

- `installation.md`: Installation and environment setup.
- `features.md`: Capability matrix and implementation status.
- `development.md`: Local development and contribution workflow.
